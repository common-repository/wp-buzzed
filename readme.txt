== Description ==

WP-Buzz is a plugin to put a post to your [Buzz] account everytime you publish a blog post.
All you have to do is setup this plugin, enter your sitename,oauth_consumer_key,oauth_consumer_secret and your domain.

== Installation ==

1. Extract the wp-buzzed.zip
2. Upload `wp-buzzed` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to `Settings` menu, select the `wp-buzz` menu
5. Enter your sitename,oauth_consumer_key,oauth_consumer_secret and your domain.
